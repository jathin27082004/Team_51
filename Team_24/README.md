## Project Description
GesturePlay is a gesture-controlled learning platform designed for children with autism. 
It uses hand gestures, visual games, and interactive quizzes to make learning engaging, accessible, and inclusive.

## Features
- Gesture-based interaction
- Emotion Quiz
- Math Quiz
- Face Matching Game
- Sketch Shape Game
- Color Match Game
- Memory Game
- Gesture Simon Says
- Progress Tracking
